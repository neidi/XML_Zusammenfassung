Datenzentrische XNL Dokumente haben viele kleine Datenitems, die einer spezifischen Struktur folgen.

Beispiel: Config File

\begin{lstlisting}
<folder name="C:">
	<folder name="Windows">
		<folder name="system32">
			<files>
				<file name="fsutil.exe" />
				<file name="ftp.exe" />
			</files>
		</folder>
	</folder>
</folder>
\end{lstlisting}
